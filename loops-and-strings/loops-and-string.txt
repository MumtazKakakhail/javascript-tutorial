loops
loops are used to execute a piece of code again and again.
1- for-loop
2- infinite-loop => A loop that never ends.
3- while-loop
4- do-while loop
5- for-of loop
6- for-in loop

Strings
String is a sequence of characters used to represent text.
1- create String
let str = "hello!";
2- string length
str.length
3- string indices or index
str[0],str[1],str[2]

Template literals in  js 
A way to have embadded expression in Strings
`this is a template literal`
String Interpolation
To create string  by doing substitution of placeholders
e:g: `string text ${expression} string text`

String Methods
There are built-in functions to manipluate a string.
str.toUpperCase()
str.toLowerCase()
str.trim() => to remove whitespaces
str.slice(start, end?) => return part of string
str1.concat(str2) => join str2 with str1
str.replace(searchVal, newVal)
str.charAt(idx)
